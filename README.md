# Grade Calculator

Open source tool for students and teachers to use to calculate all the different types of grades.

## Qualification type covered

- A Levels
- BTEC (NQF)
- BTEC (QCF)
- BTEC (RQF)
- Cambridge PREU
- Core maths
- OCR Cambridge Technical
- T Levels
